Git 常用命令

## `git clone`

从git服务器拉取代码

### `git config`

git config user.name name
git config user.email email
配置开发者用户名和邮箱

### `git branch`

创建、重命名、查看、删除项目分支，通过 Git 做项目开发时，一般都是在开发分支中进行，开发完成后合并分支到主干。

### `git checkout`

切换分支

### `git status`

查看文件变动状态

### `git commit`

提交文件变动到版本库

### `git push`

将本地的代码改动推送到服务器

### `git pull`

将服务器上的最新代码拉取到本地

### `git log`

查看版本提交记录

### `git tag`

为项目标记tag
